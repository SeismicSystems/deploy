# Testnet

## Network Configuration

| Property | Value |
|----------|-------|
| **Network Name** | Seismic Testnet |
| **Chain ID** | 5124 (0x1404) |
| **Chain Type** | EVM L1 |
| **RPC HTTP** | https://internal-testnet.seismictest.net/rpc |
| **RPC WS** | wss://internal-testnet.seismictest.net/ws |
| **Block Rate** | 1 block per ~600ms |
| **Finality** | 1 block (may become 2, but still < 2 seconds) |

:::info
The testnet RPC endpoints currently have no uptime guarantee.
:::
