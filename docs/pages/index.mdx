# Seismic Network Documentation

Welcome to the Seismic Network documentation. This guide covers everything you need to know about running nodes, building applications, and integrating with the Seismic Network.

## About Seismic

Seismic is an EVM-compatible Layer 1 blockchain that introduces shielded (private) computation capabilities to Solidity smart contracts. Built on a fork of Reth (the Rust Ethereum client), Seismic enables privacy-preserving transactions while maintaining full compatibility with the Ethereum ecosystem.

**Key Features:**
- **EVM-Compatible**: Deploy existing Solidity contracts without modification
- **Shielded Computation**: Private data storage and computation via new opcodes
- **Fast Finality**: 1-2 blocks (< 2 seconds) to finality
- **Intel TDX**: Secure enclaves for trusted execution
