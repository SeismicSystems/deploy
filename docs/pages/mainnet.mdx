# Mainnet

## Network Configuration

| Property | Value |
|----------|-------|
| **Network Name** | Seismic |
| **Chain ID** | 5123 (0x1403) |
| **Chain Type** | EVM L1 |
| **Block Rate** | 1 block per ~600ms - 1s |
| **Finality** | 1 or 2 blocks (< 2 seconds) |
| **Gas Token** | Seismic (native token) |

## Genesis Date

The mainnet genesis date will be announced publicly. Follow official channels for updates:
- **Twitter**: Seismic announcements
- **Discord**: Community updates
- **Website**: https://seismic.systems

## Gas Pricing

Seismic uses the same gas pricing model as Ethereum with all latest EIPs activated, including EIP-1559 dynamic transaction fees. The minimum gas price defaults to 7 gwei (Reth default), but actual prices vary based on network activity.

## Block Explorer

Block explorer URLs will be announced closer to mainnet launch.
